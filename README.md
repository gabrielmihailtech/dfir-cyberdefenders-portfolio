# DFIR CyberDefenders Portfolio

This repository contains hands-on Digital Forensics and Incident Response (DFIR) investigations completed through CyberDefenders labs.

The purpose of this portfolio is to demonstrate practical skills in analyzing cyber attacks, identifying indicators of compromise (IOCs), and reconstructing attacker behavior using real-world scenarios.

---

## 🔍 Areas Covered

- Network Forensics (Wireshark – HTTP & SMB analysis)
- Threat Intelligence (VirusTotal, Any.Run, malware analysis)
- Memory Forensics (Volatility)
- Mobile Forensics (Android artifacts, ALEAPP)
- OSINT (Open Source Intelligence investigations)
