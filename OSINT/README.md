# OSINT (Open Source Intelligence)

This section contains investigations based on publicly available information to identify and analyze potential threats.

The lab included in this section covers:

- Discovery of exposed credentials in public repositories
- Correlation of online identities across platforms (GitHub, Instagram)
- QR code analysis
- Image-based geolocation techniques
- Tracking user activity and movement using social media

The investigation demonstrates how open-source data can be used to identify insider threats, reconstruct behavior, and uncover sensitive information without direct system access.
