# Mobile Forensics

This section contains investigations focused on analyzing mobile device data, specifically Android artifacts.

The lab included in this section covers:

- Extraction and analysis of application data (APK files)
- Examination of SMS databases using SQLite
- Reconstruction of user activity and communication patterns
- Analysis of location data from Google Maps
- Investigation of application storage (Discord, cached files, SQLite WAL)
- Recovery of hidden or fragmented data from mobile applications

The investigation demonstrates the ability to extract and correlate data from multiple mobile artifacts to reconstruct user behavior and identify key evidence.
