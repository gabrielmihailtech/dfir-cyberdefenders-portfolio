# Network Forensics

This section contains investigations focused on analyzing network traffic using tools such as Wireshark.

The labs included in this section cover:

- HTTP traffic analysis and web server exploitation
- SMB protocol analysis and lateral movement detection
- Credential extraction from network traffic
- Identification of attack patterns such as port scanning and enumeration
- Reconstruction of full attack chains (from initial access to persistence)

The investigations demonstrate practical experience in detecting, analyzing, and understanding real-world network-based attacks.
