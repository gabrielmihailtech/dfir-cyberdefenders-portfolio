# Threat Intelligence

This section contains investigations focused on malware analysis and threat intelligence using platforms such as VirusTotal and Any.Run.

The labs included in this section cover:

- Identification of malware families and behavior
- Extraction of Indicators of Compromise (IOCs)
- Analysis of command-and-control (C2) infrastructure
- Malware configuration analysis (e.g. encryption keys, dropped files)
- Mapping behaviors to MITRE ATT&CK techniques

The investigations demonstrate the ability to analyze malware artifacts, understand attacker techniques, and correlate threat intelligence data from multiple sources.
